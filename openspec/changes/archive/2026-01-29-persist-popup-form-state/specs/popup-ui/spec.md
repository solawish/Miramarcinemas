## ADDED Requirements

### Requirement: Popup 表單狀態持久化
Extension SHALL 在 popup 關閉後重新開啟時，保留使用者上次選擇的電影、時間、票數以及票種關鍵詞輸入框的內容。

#### Scenario: 關閉重開後還原電影與票數、票種關鍵詞
- **WHEN** 使用者曾選擇電影、票數或輸入票種關鍵詞，關閉 popup 後再次開啟
- **THEN** popup 載入完成且電影/場次資料就緒後，電影下拉選單、票數下拉選單與票種關鍵詞輸入框應還原為上次儲存的值（電影、票數僅在目前選項中存在該值時還原）

#### Scenario: 關閉重開後還原時間選單
- **WHEN** 使用者曾選擇電影與場次時間，關閉 popup 後再次開啟
- **THEN** 在還原電影選擇並依此更新時間下拉選單後，若上次儲存的時間 value 存在於當前時間選項中，應將時間下拉選單還原為該值；若不存在（例如場次已變更或下架），則不強制還原，保留「請選擇時間」或空選

#### Scenario: 選擇或輸入變更時寫入儲存
- **WHEN** 使用者變更電影、時間或票數下拉選單，或變更票種關鍵詞輸入框內容
- **THEN** Extension 應將當前四個欄位的值寫入持久化儲存（例如 `chrome.storage.local`），供下次開啟 popup 時還原使用
